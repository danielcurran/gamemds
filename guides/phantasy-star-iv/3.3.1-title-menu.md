### 3.3.1. Title Menu

<!-- MIXED -->

| o CONTINUE |  |  |  |

| o START |  | o START |  |
| o ERASE DATA |  |  |  |

The Title Menu appears after the initial introduction sequence and Title Screen. It has either one or three options:

START - Select to begin a Phantasy Star IV game from scratch.

CONTINUE - Select to load a saved game from one of the three files.

ERASE DATA - Select to delete a saved game from one of the three files.
