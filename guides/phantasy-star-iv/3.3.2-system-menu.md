### 3.3.2. System Menu

| o SAVE |  |
| --- | --- |
| o MESSAGE |  |
| o BTL SPD |  |
| o BUTTONS |  |

The System Menu is accessed by pressing START while in the field. It allows you to change some of the settings and also to save the game.

SAVE - Displays the three save files. Selecting one of the files allows you to write or overwrite save data, meaning you can restore the game at a later date. Can only be used in towns or outside.

MESSAGE - Allows you to set the message speed, from 1 (Fast) to 5 (Slow).

BTL SPD - Allows you to set the battle speed, from 1 (Fast) to 5 (Slow).

BUTTONS - Allows you to choose the configuration of the A, B and C buttons, from all six possibilities.
