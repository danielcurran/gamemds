# 3. Controls

This section is here because its common in the majority of peoples FAQs; if you don't know the controls of a Game, how do you expect to survive. Anyway, I will explain briefly about what each button on your wonderful SNES controller can do so read on and be amazed. Please Note that this is a good section to read just before the walkthrough it would make life much easier for you in the long run of this beautiful, exciting game!

"A" BUTTON -> Used for almost everything, this will allow you to interact with the game, in other words, its used for attacking, casting spells and skipping through dialog as you read it. Its the opposite to the "B" button too, but this is the button that I tend to press frequently all the way through this game.

"B" BUTTON -> If you need to get out of certain menus, you should try using the "B" button; you will need it in battles and in shops. For example you are fighting against "Gades" and Guy is ready to do an Ikari technique such as the Kukri's "Thunder Blast" attack; but in this battle you don't want to use your IP's up. You have realized that you want to attack normally; if you were to press the "B" button, you could go back to the previous menu; this is a helpful button during the game, because like the "A" button it gets used a lot.

"X" BUTTON -> If you were to press our beloved "X" button; you would be able to access the "Status Screen" and equip weapons and armours, or even cast healing spells outside of battle. This comes to your aid at the best of times; I mean how can Maxim equip the legendary Duel Blade without being able to access the status screen.

"Y" BUTTON -> Well, you will only use this in Caves, Towers and Dungeons; this button is used for firing "Skill Arrows" and various other Skill Items. This button is only used occasionally, but it would be a good idea if you knew its exact purpose; use this for blowing up weak walls; with bombs; like in "Zelda"!

"SELECT" -> Opens up the Skill menu, so you can change your skills. Say you wanted to use the arrow skill and you were currently using some bombs; you would be able to select the skill from this menu.

"START" -> This is stupid, you would maybe use this to interact the odd time here and there. Not used commonly at all, perhaps you could press this button to start the game.

"R" BUTTON -> After picking up a vase, you can hold down this button, so that you can turn to any direction, without moving a step. This button is also used to make a spell hit all enemies on the screen; which tends to be pretty useful, and it can also be used for making a healing spell hit all allies too.

"L" BUTTON -> Finally we have the "L" button, which is used for a few different things; which can be done with all of the other buttons. Inside a dungeon, it does the same as the "Y" button; in other words, you can use this to fire arrows; and blow things up with bombs. This button can also be used for confirming moves in a battle, so you also have a replacement for the "A" button; but it cannot be used for activating menu items; or equipping your characters!
