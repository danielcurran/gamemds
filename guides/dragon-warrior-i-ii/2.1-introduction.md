## 2.1. Introduction

Alright! The first ever Dragon Quest game! Aren't you excited!? In case you didn't read the NOTE above, I'll write it again: This guide was written for an English Fan Translation of the Game which was made by RPGOne. I wrote this guide using the latest version of the Fan Translation (2.0) which was released on the 23th of June, 2003. So yeah! This guide is made specifically for that Fan Translation but I'm sure you can use it for the other versions of the game. Anyway, let's begin! Start up the game. The first thing you'll be shown is RPGOne and the names of the people who translated this game. Great big thanks to them. Press the START Button. After you see the creators of the game, you'll be taken to the Title Screen. You get to Select Dragon Quest I or II. Select Dragon Quest I and let's begin. Select Begin A New Adventure. Select your Adventure Log then input your character name. I named mine Hero. Next, you get to select the Message Speed. I went for 5. The game will begin.
