### 15.4.2. Lashiec

```
Met: Air Castle Basement        Recommended Level before fighting:
  HP: 6383                         Chaz: 36-37    Wren: 34-35
 EXP: 29999                        Rika: 39-40    Kyra: 34-35
 MST: 1                            Rune: 34-35


In the first round, get Chaz to use RAYBLADE, have Rika use SANER, let
Rune throw out an EFESS, have Wren set up a BARRIER, and get Kyra to use
WARLA. This should set up a decent defence and also start the offensive
against Lashiec.

Lashiec will use one of a few different Skills to begin with - either
THNDHALBRT or ANOTHRGATE (both of which are powerful hit-all Skills, the
latter being the most damaging), or POSSESSION (a single-target Skill that
inflicts Sleep on one character). In the following rounds, have Chaz keep
using RAYBLADE (or NATHU when this runs out), get Rika to use GISAR or
NARES (depending on who was damaged in the round before), get Rune to
continue using EFESS (or NAFOI when this runs out), let Wren just attack
(or use RECOVER if he needs HP), and keep Krya using NAFOI (or GIRES or
a curative item if you need a little extra healing).

Once Lashiec gets to 1/4 HP, he will use REINFORCE, which increases all of
his statistics. This makes him that extra bit tougher, but indicates that
he's almost down. Keep using the above strategy, putting a little more
emphasis on healing if you need it, or letting Rika use DBLSLASH if there
is no healing to be done.

If any of your characters go down, get a fast character to use REVER or a
MOON-DEW, and follow this up with a slower character using GIRES or a
DIMATE or TRIMATE. Alternatively, just use a SOL-DEW or a REPAIR-KIT.

Keep pummelling Lashiec and maintaining your party's HP levels, and this
ancient king will be dethroned in about six or seven rounds.
```
