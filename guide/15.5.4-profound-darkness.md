### 15.5.4. Profound Darkness

```
Met: The Edge                   Recommended Level before fighting:
  HP: 4120/4620/12999              Chaz: 49+         Wren: 46+
 EXP: N/A                          Rika: 52+        Other: 45+
 MST: N/A                          Rune: 45+


The final boss battle of the game consists of three stages, each involving
combat with one of the incarnations of The Profound Darkness. They all
differ in their strength and abilities, so this strategy guide will explain
each incarnation separately. Also, the fifth character you have chosen will
determine the best method of fighting The Profound Darkness, so there are
five different strategies representing each possible choice. Good luck!

Stage 1 - The first incarnation is fairly unsettling it its appearance, but
          it is the weakest of the three. It will use only single-target
          attacks, including RAY-BREATH, FIREBREATH and SHDWBREATH. They
          are fairly similar in their strength, but certain characters will
          be weaker or more resistant to certain attacks. You will only
          need to use single-target healing to overcome these Skills.

   Hahn - In the first round, have Chaz use RAYBLADE, Rika use SANER to
          speed up the party, Rune use NAWAT, Wren throw up a BARRIER, and
          Hahn use his VISION Skill.

          In the second round, get Chaz to use RAYBLADE again, let Rika use
          DEBAN, have Rune use NAWAT, have Wren just attack or use RECOVER,
          and let Hahn use GIRES or NARES (or NAWAT if Wren uses RECOVER).

          In the following rounds, keep Chaz using RAYBLADE, have Rika take
          over the healing (or use DBLSLASH if no healing is required), let
          Rune continue using NAWAT, get Wren to attack or use RECOVER, and
          have Hahn use NAWAT as well.

          Keep using this strategy until the incarnation dissipates and the
          second form appears, which should take no more than six rounds.

   Gryz - In the first round, have Chaz use RAYBLADE, Rika use SANER to
          speed up the party, Rune use NAWAT, Wren throw up a BARRIER, and
          Gryz perform a WAR CRY.

          In the second round, get Chaz to use GIRES or NARES (or RAYBLADE
          if Wren uses RECOVER), let Rika use DEBAN, have Rune use NAWAT,
          have Wren just attack or use RECOVER, and let Gryz just attack.

          In the following rounds, get Chaz to use RAYBLADE, have Rika take
          over the healing (or use DBLSLASH if no healing is required), let
          Rune continue using NAWAT, get Wren to attack or use RECOVER, and
          have Gryz attack as well.

          Keep using this strategy until the incarnation dissipates and the
          second form appears, which should take no more than six rounds.

   Demi - In the first round, have Chaz use RAYBLADE, Rika use SANER to
          speed up the party, Rune use NAWAT, Wren attack normally, and
          Demi set up a BARRIER.

          In the second round, get Chaz to use GIRES or NARES (or RAYBLADE
          if Wren uses RECOVER), let Rika use DEBAN, have Rune use NAWAT,
          have Wren just attack or use RECOVER, and let Demi just attack or
          use RECOVER too.

          In the following rounds, get Chaz to use RAYBLADE, have Rika take
          over the healing (or use DBLSLASH if no healing is required), let
          Rune continue using NAWAT, get Wren to attack or use RECOVER, and
          have Demi attack or use RECOVER as well.

          Keep using this strategy until the incarnation dissipates and the
          second form appears, which should take no more than six rounds.

   Raja - In the first round, have Chaz use RAYBLADE, Rika use SANER to
          speed up the party, Rune use NAWAT, Wren throw up a BARRIER, and
          Raja use BLESSING.

          In the second round, get Chaz to use RAYBLADE, let Rika perform a
          DBLSLASH, have Rune use NAWAT, have Wren attack or use RECOVER,
          and let Raja use RES or GIRES (or defend if Wren uses RECOVER).

          In the following rounds, get Chaz to use RAYBLADE, have Rika keep
          using DBLSLASH, let Rune continue using NAWAT, get Wren to attack
          or use RECOVER, and have Raja continue the healing.

          Keep using this strategy until the incarnation dissipates and the
          second form appears, which should take no more than six rounds.

   Kyra - In the first round, have Chaz use RAYBLADE, Rika use SANER to
          speed up the party, Rune use NAWAT, Wren throw up a BARRIER, and
          Kyra use her WARLA Skill.

          In the second round, get Chaz to use RAYBLADE, let Rika use GIRES
          or NARES (or DBLSLASH if no healing is required), have Rune use
          NAWAT, have Wren attack or use RECOVER, and let Kyra use NAFOI.

          In the following rounds, get Chaz to use RAYBLADE, have Rika keep
          up the healing or using DBLSLASH, let Rune continue using NAWAT,
          get Wren to attack or use RECOVER, and have Kyra use NAFOI.

          Keep using this strategy until the incarnation dissipates and the
          second form appears, which should take no more than six rounds.

Stage 2 - The second incarnation has only slightly more HP than the first,
          but has far more powerful Skills. Apart from a lightning-style
          normal attack, the second form has use of DISTORTION, LIGHTSHOWR
          and ANOTHRGATE. These three Skills deal a considerable amount of
          damage to all members of the party, so target-all healing will be
          the order of the day this time. As there is no initial setup to
          be done, the second stage shouldn't last as long as the first.

   Hahn - From the beginning, keep Chaz using RAYBLADE (or CROSSCUT when
          this runs out), have Rika use GIRES or GISAR (or use DBLSLASH if
          no healing is required), let Rune continue using NAWAT, get Wren
          to attack or use RECOVER, and have Hahn use NAWAT as well.

          Keep using this strategy until the incarnation dissipates and the
          final form appears, which should take no more than five rounds.

   Gryz - From the beginning, keep Chaz using RAYBLADE (or CROSSCUT when
          this runs out), have Rika use GIRES or GISAR (or use DBLSLASH if
          no healing is required), let Rune continue using NAWAT, get Wren
          to attack or use RECOVER, and have Gryz just attack as well.

          Keep using this strategy until the incarnation dissipates and the
          final form appears, which should take no more than five rounds.

   Demi - From the beginning, keep Chaz using RAYBLADE (or CROSSCUT when
          this runs out), have Rika use GIRES or GISAR (or use DBLSLASH if
          no healing is required), let Rune continue using NAWAT, have Wren
          attack or use RECOVER, and get Demi to use PHONOMEZER or RECOVER.

          Keep using this strategy until the incarnation dissipates and the
          final form appears, which should take no more than five rounds.

   Raja - From the beginning, keep Chaz using RAYBLADE (or CROSSCUT when
          this runs out), have Rika use DBLSLASH, let Rune continue using
          NAWAT, get Wren to attack or use RECOVER, and have Raja keep up
          his expert healing by using a RES or a SAR (or else just defend).

          Keep using this strategy until the incarnation dissipates and the
          final form appears, which should take no more than five rounds.

   Kyra - From the beginning, keep Chaz using RAYBLADE (or CROSSCUT when
          this runs out), have Rika use GIRES or GISAR (or use DBLSLASH if
          no healing is required), let Rune continue using NAWAT, get Wren
          to attack or use RECOVER, and have Kyra keep using NAFOI.

          Keep using this strategy until the incarnation dissipates and the
          final form appears, which should take no more than five rounds.

Stage 3 - The third and final incarnation of The Profound Darkness is the
          toughest of them all, and with about three times as many HP it
          will definitely take the longest. It has a standard attack which
          is fairly damaging, and a few other abilities as well. These
          consist of POSSESSION (that may inflict Sleep on a character),
          CANCELING (extremely annoying, as it removes all assistance
          effects from the party such as BARRIER, DEBAN, SANER, etc.), and
          the powerful MEGID Technique. Don't hold back. This is it.

   Hahn - From the beginning, have Chaz use MEGID (or CROSSCUT if you can't
          use this Technique), get Rika to take care of the healing (or use
          DBLSLASH if healing isn't required), let Rune throw out a LEGEON
          (or EFESS when this runs out), get Wren to use POSIBOLT if he has
          it (else attack, using RECOVER when needed), and have Hahn use
          his ASTRAL Skill (or NAWAT when the stock is depleted).

          After CANCELING, have Rika use DEBAN instead of healing, let Wren
          use BARRIER instead of attacking or healing, and get Hahn to
          cover the curative responsibility by using NARES or an item.

          If Rika runs out of TP from healing, have Hahn take over by using
          Techs, or by dipping into your (hopefully extensive) item supply.

          Keep using this strategy until The Profound Darkness is finally
          destroyed, which should between nine to twelve rounds.

   Gryz - From the beginning, have Chaz use MEGID (or CROSSCUT if you can't
          use this Technique), get Rika take care of the healing (or use
          DBLSLASH if healing isn't required), let Rune throw out a LEGEON
          (or EFESS when this runs out), get Wren to use POSIBOLT if he has
          it (else attack, using RECOVER when needed), and have Grzy keep
          attacking (this is his forte).

          After CANCELING, have Rika use DEBAN instead of healing, let Wren
          use BARRIER instead of attacking or healing, and get Gryz to
          cover the curative responsibility by using appropriate items.

          If Rika runs out of TP from healing, have Gryz take over the job
          by dipping into your (hopefully extensive) item supply.

          Keep using this strategy until The Profound Darkness is finally
          destroyed, which should between nine to twelve rounds.

   Demi - From the beginning, have Chaz use MEGID (or CROSSCUT if you can't
          use this Technique), get Rika take care of the healing (or use
          DBLSLASH if healing isn't required), let Rune throw out a LEGEON
          (or EFESS when this runs out), get Wren to use POSIBOLT if he has
          it (else attack, using RECOVER when needed), and have Demi use
          PHONOMEZER if she can (else attack, using RECOVER when needed).

          After CANCELING, have Rika use DEBAN instead of healing, let Wren
          use BARRIER instead of attacking or healing, and get Demi to
          cover the curative responsibility by using MEDIC PW or an item.

          If Rika runs out of TP from healing, have Demi take over with her
          MEDIC PW or by utilizing your (hopefully extensive) item supply.

          Keep using this strategy until The Profound Darkness is finally
          destroyed, which should between nine to twelve rounds.

   Raja - From the beginning, have Chaz use MEGID (or CROSSCUT if you can't
          use this Technique), get Rika to keep using DBLSLASH (or attack
          when the stock is exhausted), let Rune throw out a LEGEON (or
          EFESS when this runs out), get Wren to use POSIBOLT if he has
          it (else attack, using RECOVER when needed), and have Raja keep
          using curative Techs (or ST. FIRE if no healing is required).

          After CANCELING, have Rika use DEBAN instead of attacking, and
          let Wren use BARRIER instead of attacking or healing.

          If Raja runs out of TP from healing, have Rika take over for a
          round while Raja uses ATARAXIA on himself to regain some TP.

          Keep using this strategy until The Profound Darkness is finally
          destroyed, which should between nine to twelve rounds.

   Kyra - From the beginning, have Chaz use MEGID (or CROSSCUT if you can't
          use this Technique), get Rika take care of the healing (or use
          DBLSLASH if healing isn't required), let Rune throw out a LEGEON
          (or EFESS when this runs out), get Wren to use POSIBOLT if he has
          it (else attack, using RECOVER when needed), and have Kyra use
          TANDLE (or NAFOI if this runs out).

          After CANCELING, have Kyra use WARLA instead of attacking, and
          let Wren use BARRIER instead of attacking or healing.

          If Rika runs out of TP from healing, have Kyra take over with her
          Techs or by dipping into your (hopefully extensive) item supply.

          Keep using this strategy until The Profound Darkness is finally
          destroyed, which should between nine to twelve rounds.
```
