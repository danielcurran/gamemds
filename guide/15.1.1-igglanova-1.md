# 15. Boss Strategies
<a id="s15-1"></a>
## 15.1. Act I

<a id="s15-1-1"></a>

### 15.1.1. Igglanova 1

```
Met: Academy Basement           Recommended Level before fighting:
  HP: 300                          Alys: 7
 EXP: 173                          Chaz: 2-3
 MST: 54                           Hahn: 2-3


To defeat the Igglanova quickly, have Alys use her VORTEX skill and Chaz
and Hahn attack in the first round. The Igglanova will spawn a Xanafalgue
every round until there are two on the screen, at which point it will
attack, so make sure that Chaz attacks the Xanafalgue each round and keep
having Alys use VORTEX. Hahn should be attacking whatever necessary
(usually Igglanova) or using RES if anyone needs healing (which shouldn't
happen). Igglanova should go down in four or five rounds.
```
