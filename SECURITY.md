# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately by emailing
[daniel@curran.io](mailto:daniel@curran.io).

**Do not open a public issue —** please give us time to address it before
disclosure.

You should receive a response within 48 hours.

## Scope

Security-relevant issues include:

- CSP bypass or XSS in the reader app
- Path traversal in guide file loading
- Malicious content injection via walkthrough markdown files

## Supported Versions

Only the latest commit on the `main` branch receives security patches.
